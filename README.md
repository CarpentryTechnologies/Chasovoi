Часовой это SCADA-система верхнего уровня, написанная на Delphi 7. Инструментальная программа для автоматизации контроля и управления технологическим процессом в режиме реального времени. Основная цель Часового – дать оператору, управляющему технологическим процессом, полную информацию об этом процессе и необходимые средства для воздействия на него. Заточена под электронику компании Столярные Технологии г.Тольятти, учредитель Катунов Юрий Юрьевич г. Тольятти.
